Курс валют

1 USD = ... UAH 1 EUR = ... UAH

Інструкція:

  1. Конвертування відбувається між такими валютами: EUR (євро), UAH (гривня), USD (долар).
  2. Курс валюти зображений ліворуч в кутку, приходить з публічного API Національного банку України.
  3. Програма конвертує валюту "From"-"To", також у зворотньому порядку "To"-"From".
  4. Конвертування відбувається автоматично при вводі число в "From" або "To", результат відображений у протележному input, тобто "To" або "From" відповідно.
  5. При вводі від'ємних чисел програма повертає 0 у протележний input.
  6. При зміні валюти в будь-якому section в обох input виводиться 0.
  7. Результат заокруглюється до другого знака після коми.
